# Learning Objectives

By the end of the PACE flow, the student should be able to:

1. State the case decision in plain language.
2. Identify the relevant data and missing information.
3. Explain the finance concept from intuition to formula.
4. State and defend key assumptions.
5. Build or debug the required model logic.
6. Explain the recommendation, caveats, and tradeoffs for class discussion.
