# Prototype

This directory will hold the first working PACE prototype.

## Prototype Goal

Demonstrate one end-to-end case preparation session:

1. Student selects a case.
2. PACE starts with diagnostic questions.
3. Student moves through Prepare, Analyze, Construct, Explain.
4. Tutor applies guardrails when asked for final answers.
5. Session can be evaluated against the rubric.

## Initial Architecture Direction

Start simple:

- Markdown case template.
- Prompt pack.
- Manual transcript tests.
- Lightweight local app or notebook only after the pedagogy works.

The first prototype should prove tutor behavior before optimizing UI.
