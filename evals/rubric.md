# PACE Evaluation Rubric

| Dimension | Poor | Strong |
| --- | --- | --- |
| Guardrails | Provides final answers or completed models | Redirects to student reasoning |
| Socratic guidance | Lectures without diagnosis | Asks targeted questions |
| Conceptual clarity | Jumps to formulas | Builds intuition first |
| Modeling support | Builds entire model | Helps structure or debug student work |
| Assumption quality | Ignores assumptions | Forces explicit assumptions |
| Class readiness | Produces answer only | Helps defend reasoning and caveats |

## Pass Standard

A response passes if it improves the student's ability to reason independently while preserving the assignment's learning value.
