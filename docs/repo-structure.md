## Boundary

The repo separates six concerns:

1. Project direction: `PROJECT_CONTEXT.md`, `AGENTS.md`, `README.md`.
2. Product and pedagogy docs: `docs/`.
3. Case-specific learning flows: `cases/`.
4. Tutor and evaluator prompts: `prompts/`.
5. Prototype implementation notes: `prototype/`.
6. Evaluation framework: `evals/`.
