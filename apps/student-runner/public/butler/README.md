# Butler Local Assets

This directory is intentionally excluded from public source control for case-material handling.

To run the Butler prototype with the actual course files, add these local files:

- `Butler_Day_1_Tools.pdf`
- `Butler_Lumber_Day_1_Template.xlsx`

The app expects them at:

- `/butler/Butler_Day_1_Tools.pdf`
- `/butler/Butler_Lumber_Day_1_Template.xlsx`
